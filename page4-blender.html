3D Modeling Blender Dungeon

Project Overview

I took inspiration for my dungeon from jails. My dungeon consists of a hallway with jail cells on either side. At the end of the hallway is a treasure room with an exit (a ladder leading up), but before that there’s a spike trap pit.

  • Used various modifers to speed up workflow.
  • Created single floor and wall tiles, then applied the Array modifier on both the X and Y axes to duplicate and align them automatically instead of placing each one manually.​
  • Used the Boolean modifier on both the dungeon and assets as a cutting tool for the jail cell lock, pillar arch, and much more
  • Tools like scaling, extrude, inset, bevel, to add more detail and patterns
